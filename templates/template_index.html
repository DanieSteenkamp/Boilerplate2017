Boilerplate2017 - A Generic Starter Pack for Bulk Ad Manager

Boilerplate2017 is a starter pack and the core framework to quickly build sites in Bulk Ad Manager without having to build everything from the ground up every time.

Overview:

  • Built on Bootstrap 3 with Font Awesome already included.
  • The build includes various commonly used components such as sliders, top lists, star ratings, modal windows, accordions etc.. all included for easy reference as a snippet.
  • This build is unstyled and any new sites should still be styled uniquely; this process however is helped along by using Sass and Gulp to streamline the styling process and keep things tidy and uniform.
  • All components and layouts are already fully mobile responsive across all device screen sizes.
Note: The Boilerplate2017 build is primarily meant for portal / brand sites and shouldn't be utilized for network sites.
